# MoodMeal AI 🍔🎬🎵

MoodMeal AI is an AI-powered web application that analyzes user emotions from text input and provides personalized recommendations across food, movies, and music.

## 🚀 Features
- Emotion-based mood detection
- Multi-language movie recommendations (English, Tamil, Korean)
- Personalized suggestions for:
  - 🍔 Food
  - 🎬 Movies
  - 🎵 Music
- Responsive and modern UI
- Clickable real-world links (Google, Spotify)

## 🛠 Tech Stack
- Frontend: React.js, Tailwind CSS
- Backend: Flask / Node.js
- API: REST API

## 💡 How it Works
1. User enters mood
2. System detects emotion using keyword logic
3. Recommendations are generated based on mood and language
4. User can directly access results via external links

## 📌 Future Improvements
- Real-time API integration (TMDB, Spotify)
- Voice-based mood detection
- User login and history tracking
